# Kcarnold Tap

## How do I install these formulae?

`brew install kcarnold/tap/<formula>`

Or `brew tap kcarnold/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kcarnold/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
